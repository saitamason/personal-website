Remix Expenses: Empowering Personal Expense Management

Remix Expenses is a full-stack application that enables efficient management of personal expenses, including adding, removing, and analyzing data. The application boasts JSON Web Token (JWT) authentication and relies on a MongoDB database. Combining server-side rendering (SSR) with single-page application (SPA) functionality ensures both performance and convenience.

By creating this application, I acquired in-depth knowledge of the Remix framework, from routing pages and implementing server-side code to handling errors and styling with CSS. Moreover, the project allowed me to sharpen my TypeScript and React skills.

Natio: An Educational SPA with MVC Architecture

Natio is a personal project designed from scratch, offering users an educational experience by exploring countries and their neighbors, saving favorites, and tracking the progress of discovered nations.

This single-page app (SPA) is written in vanilla JavaScript, HTML, and Sass (SCSS), following the model view controller (MVC) architecture, and efficiently fetches data from the REST Countries API. With a fully responsive design, Natio guarantees compatibility with mobile devices. One of its unique visual features is the dynamically changing background color, calculated based on the colors of each country's flag. Moreover, I implemented a custom search functionality, enabling users to easily find countries without needing to input the whole name or diacritics.

Developing Natio allowed me to master the process of creating an MVC application with vanilla JS, providing valuable experience outside the convenience of using a framework like React.

Natours: A Full-Stack App with Authentication, Authorization, and CRUD Operations

Natours is a comprehensive full-stack application created as part of a course, offering authentication, authorization, and CRUD operations for managing trips on behalf of a fictional traveling company. This ambitious project incorporates Node.js, Express.js, MongoDB, and Mongoose. Several libraries, such as helmet, jsonwebtoken, axios, bcryptjs, sharp, and nodemailer, have been employed. The app also utilizes server-side rendering (SSR) with Pug.

The process of creating Natours provided me with invaluable learning opportunities, marking my first full-stack project and significantly expanding my skill set. Based on the knowledge from this project, I went on to create a Node.js back-end for my own workout application (work in progress). This exciting venture has allowed me to further delve into backend development and gain hands-on experience in building applications that handle data, authentication, and user interactions.
